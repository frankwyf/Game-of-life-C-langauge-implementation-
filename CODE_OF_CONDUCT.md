# Code of Conduct

## Our Pledge

We are committed to making participation in this project a harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive
- Assume good intent
- Give and receive feedback professionally
- Focus on technical issues, not personal attacks

## Unacceptable Behavior

- Harassment, insults, or discrimination
- Publishing private information without consent
- Abusive or threatening language
- Repeated disruptive behavior

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, and issues that violate this Code of Conduct.

## Scope

This applies to project spaces, including issues, pull requests, and discussions.

## Reporting

If you experience or witness unacceptable behavior, report it to the maintainer through GitHub.
